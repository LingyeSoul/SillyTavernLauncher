## 技术栈 🧰

- 🐍 Python 3.9+ - 核心编程语言
- 🎨 Flet框架 - GUI开发
- 📦 Git - 版本控制
- 🌐 Node.js 18+ - 服务运行

## 快速开始 🚀

```bash
# 安装依赖
pip install flet ruamel.yaml packaging pystray aiohttp
```

## 功能特性 🌟

### 环境管理
- 自动检测系统环境变量
- 支持内置环境模式（env目录）

### 服务控制
- 优雅启动/停止服务
- 实时终端日志输出

### 一键更新
- GitHub检测更新
- 支持镜像源切换

### 系统托盘集成
- 最小化到系统托盘
- 托盘菜单快速操作
- 后台运行SillyTavern
